Program Lendit Version 1.0 5/1/2018

Usage Notes:
-------------------------------------------------------------------------------
- This application tracks the user's lending and borrowing history, based on the
transactions that they make with other users. This app's main feed displays all 
the transactions that a user has been involved in. 

- This application has a local calendar that stores the dates that items need to
be returned to their owner


Account Requirements:
-------------------------------------------------------------------------------
- a user must have a Facebook login, and can make a unique username after they 
first login with our app

- if someone wants to add the return event to their Google Calendar, they can sync the
event to their google account

- requires network connection to access other users and to access personal feed, 
which is pulled from firebase.
